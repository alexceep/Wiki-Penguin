# 🐧 Wiki Penguin

Uma wiki interativa inspirada no universo de **Club Penguin**, desenvolvida para reunir informações, curiosidades e conteúdos sobre o jogo de forma organizada e acessível.

---

## 📚 Sobre o projeto

O **Club Penguin Wiki** é um site criado com o objetivo de servir como uma enciclopédia digital do jogo, permitindo que usuários explorem conteúdos relacionados ao universo de Club Penguin.

O projeto inclui:
- Páginas informativas sobre o jogo
- Sistema de navegação simples
- Interface inspirada no estilo original
- Conteúdo organizado em formato de wiki

---

## 🧠 Funcionalidades

- 📖 Visualização de conteúdo estilo wiki  
- 🔍 Navegação entre páginas  
- 🎨 Interface temática inspirada em Club Penguin  
- 🧩 Estrutura organizada para expansão de conteúdo  

---

## ⚠️ Limitações atuais

Algumas funcionalidades **não estão operacionais** no ambiente do GitHub Pages:

- ❌ Sistema de login  
- ❌ Sistema de quiz  

Isso ocorre porque essas funcionalidades foram desenvolvidas utilizando **SQL (banco de dados)**, e o GitHub Pages suporta apenas sites estáticos (HTML, CSS e JavaScript), sem backend.

---
